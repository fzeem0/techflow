# Mission 3 — Log Investigation

**Difficulty:** Medium · **Estimated time:** 45 minutes

Analyze the synthetic incident in `$TECHFLOW_HOME/logs` and write the answers
to `$TECHFLOW_HOME/reports/mission-3.txt`.

1. Count the INFO, WARN, ERROR and DEBUG records in `syslog.log`.
2. Find the first and last ERROR and list the five most frequent error messages.
3. Find the ten most active IP addresses in `access.log`.
4. Count HTTP 500 responses and print their IP and URL fields.
5. Explain why `1.2.3.4` is suspicious and list its most requested URLs.
6. List the five most requested non-static URLs.
7. Find metric rows where CPU is above 85 percent.
8. Calculate average CPU usage per server.

## Verification

Use `techflow hint 3` if needed. When the report is ready, run:

```bash
techflow verify 3
```
