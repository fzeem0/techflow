# Mission 1 — Server Recon

Inspect the container without changing it. Save useful observations in
`$TECHFLOW_HOME/reports/mission-1.txt`.

1. Show the kernel, architecture, hostname and IP addresses with `uname`,
   `hostname` and `ip`.
2. Compare the one-, five- and fifteen-minute load averages with `nproc`.
3. Inspect disk usage with `df -hT` and directory sizes with `du -sh`.
4. Explain the difference between the `free` and `available` memory columns.
5. List listening ports using `ss -tulnp`.
6. Show the ten largest processes by CPU and by memory with `ps`.
7. Identify every `$TECHFLOW_HOME/data/mystery*` file using `file` and `stat`.

Run `techflow hint 1` if needed and `techflow verify 1` when finished.
