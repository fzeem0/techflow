# Mission 4 — Text Processing Gauntlet

**Difficulty:** Hard · **Estimated time:** 45 minutes

Create the requested derived files using standard text-processing tools.

1. Sort `data/ips.txt`, remove duplicates and save the result as
   `reports/unique-ips.txt`.
2. Save the ten most frequent addresses as `reports/top-ips.txt`.
3. Extract the header and all `db-01` rows from `data/metrics.csv` into
   `reports/db-metrics.csv`.
4. Produce `reports/service-errors.txt` containing ERROR counts grouped by
   service name.
5. Ensure every generated text report ends with a newline.

Useful tools: `awk`, `sed`, `cut`, `sort`, `uniq`, `head`, `wc` and `tr`.

Use `techflow hint 4` if needed. When the generated files are ready, run:
