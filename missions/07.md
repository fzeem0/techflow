# Mission 7 — Bash Automation

**Difficulty:** Hard · **Estimated time:** 60 minutes

Create executable scripts in `$TECHFLOW_HOME/scripts`.

1. `health-report` must print disk usage, available memory and load average.
2. `log-summary` must accept a log path and count INFO, WARN and ERROR lines.
3. Both scripts must use `#!/usr/bin/env bash` and `set -euo pipefail`.
4. They must quote expansions, validate arguments and return non-zero for a
   missing input file.
5. Run `bash -n` on both scripts and test success and failure paths.

## Verification

Use `techflow hint 7` if needed. To run the structural checks, use:

```bash
techflow verify 7
```
