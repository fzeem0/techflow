# Mission 8 — Git Operations Workflow

**Difficulty:** Medium · **Estimated time:** 45 minutes

The repository for this mission is `$TECHFLOW_HOME/missions/git-lab`.

1. Initialize it and configure a local training identity.
2. Commit the provided README on `main`.
3. Create a `feature/status-page` branch, add `status.txt` and commit it.
4. Return to `main` and merge the feature branch with a merge commit.
5. Create the annotated tag `training-complete`.
6. Inspect `git log --graph --oneline --all` and `git status`.

Do not modify the TechFlow source repository during this mission.
